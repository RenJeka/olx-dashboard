# Стилі та Семантичні Токени (Chakra UI v3)

В цьому проєкті використовується **Chakra UI v3**, який має потужну вбудовану систему семантичних токенів (Semantic Tokens). 

Замість того, щоб жорстко хардкодити кольори для світлої та темної теми (наприклад, `_dark={{ bg: 'gray.800' }}`), ми використовуємо ці токени. Вони автоматично генерують CSS-змінні (на зразок `var(--chakra-colors-bg-muted)`) і плавно перемикають значення при зміні теми.

## 🎨 Фони (Backgrounds - `bg`)

Використовуються для пропсів `bg` або `backgroundColor`.

* `bg.panel` — стандартний фон для карток, модальних вікон, тултіпів. Зазвичай збігається з фоном сторінки, або ледь-ледь виділяється.
* `bg.muted` — "приглушений" фон, який гарантовано дає легкий контраст від `bg.panel`. У світлій темі це блідо-сірий (`gray.100`), у темній — темно-сірий (`gray.800`). Ідеально підходить для тултіпів чи виділення блоків.
* `bg.subtle` — дуже-дуже ніжний контраст, ще менш помітний ніж muted (`gray.50` / `gray.900`).
* `bg.emphasized` — фон з підвищеним контрастом для виділення важливих блоків або кнопок.
* `bg.inverted` — інвертований фон (повністю чорний у світлій темі, повністю білий у темній).
* `bg.error`, `bg.success`, `bg.warning`, `bg.info` — стандартні фони для повідомлень про статуси.

## 🖋 Текст (Foregrounds - `fg`)

Використовуються для пропсів `color`.

* `fg.default` — основний висококонтрастний текст. Майже чорний у світлій темі, майже білий у темній. Найкращий вибір для стандартного тексту.
* `fg.muted` — другорядний текст (стандартний сірий). Добре підходить для підписів або менш важливої інформації.
* `fg.subtle` — дуже тьмяний текст (для плейсхолдерів, мікро-підписів).
* `fg.inverted` — інвертований текст (білий для чорного фону, чорний для білого фону).

## 🔲 Межі (Borders - `border`)

Використовуються для пропсів `borderColor`.

* `border.subtle` — ледь помітна лінія розділювача.
* `border.muted` — трохи більш виразна лінія для карток та інпутів.
* `border.emphasized` — яскраво виражена лінія.

---

### Приклад використання

Замість:
```tsx
<Box bg="gray.100" _dark={{ bg: 'gray.800' }} color="black" _dark={{ color: 'white' }}>
  Текст
</Box>
```

Пишіть:
```tsx
<Box bg="bg.muted" color="fg.default">
  Текст
</Box>
```
